# PolyglotMusic - Red Social Musical

## Config. inicial
1. Clonar el repo
2. Duplicar el archivo `.env.example`, renombrar la copia a `.env` y llenar las credenciales como corresponda
3. Abrir en IntelliJ IDEA, Gradle sincroniza en automatico
